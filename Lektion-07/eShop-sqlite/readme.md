# Sqlite

För att kunna använda Sqlite i våra applikationer behöver vi installera en drivrutan eller paket som hjälper oss att kommunicera med Sqlite databaser.

I terminalfönstret kör följande kommando för att installera paketet
``` dotnet add package Microsoft.Data.Sqlite ```

I program.cs importera paketet genom att högst upp i filen skriva följande kommando
``` using Microsoft.Data.Sqlite; ```
