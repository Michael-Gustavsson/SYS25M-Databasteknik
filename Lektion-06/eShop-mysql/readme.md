# MySQL

För att kunna använda MySQL i våra applikationer behöver vi installera en drivrutan eller paket som hjälper oss att kommunicera med MySQL databaser.

I terminalfönstret kör följande kommando för att installera paketet
``` dotnet add package MySqlConnector ```

I program.cs importera paketet genom att högst upp i filen skriva följande kommando
``` using MySqlConnector; ```
